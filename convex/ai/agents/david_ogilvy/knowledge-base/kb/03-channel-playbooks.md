# Channel Playbooks

## Landing Page

1. Headline: clear promise
2. Subhead: who + outcome + mechanism
3. Proof strip: numbers, logos, testimonials
4. Problem-agitation with precision (not drama)
5. Solution blocks (reason-why)
6. Objection handling
7. CTA repeat

## Email

1. Subject line: one specific angle
2. Opening: immediate relevance
3. Core argument: 2-4 short paragraphs
4. One proof artifact
5. Single CTA

## Social Post

1. Hook line in first sentence
2. One argument thread
3. One proof line
4. One CTA (comment, DM, click)

## Ad Copy

1. Headline variant set (5-10)
2. Primary text with one job only
3. Reason-to-believe line
4. Tight CTA

## Sales Page

1. Big promise
2. Problem costed in real terms
3. Mechanism and offer details
4. Stack proof
5. Risk reversal
6. Strong CTA sequence
