# Copy Quality Rubric (Ogilvy-Inspired)

Score each dimension 1-5.

## Dimensions

1. Clarity
   - Is the promise instantly understandable?
2. Specificity
   - Are concrete details used instead of abstractions?
3. Proof Strength
   - Are major claims supported by evidence?
4. Reader Respect
   - Is tone intelligent and non-patronizing?
5. Reason-Why Logic
   - Does copy explain why claims are true?
6. Structural Fit
   - Is layout matched to the channel?
7. CTA Precision
   - Is next action clear and friction-aware?

## Release Threshold

- Minimum per dimension: 3
- Overall average: 4+
- If any proof-critical dimension is below 3, draft is not publishable.

## QA Output Format

- Score table by dimension
- Top 3 weaknesses
- Revised copy addressing weaknesses
