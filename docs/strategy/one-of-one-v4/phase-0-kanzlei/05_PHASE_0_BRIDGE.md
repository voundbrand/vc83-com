## What Phase 0 IS NOT Allowed to Become

- **Not a services company.** If we're doing 20+ hours of custom work per law firm, the model is broken. Templates must reduce onboarding.
- **Not an ElevenLabs reseller.** If our only value is "we configure ElevenLabs for you," we're a consultancy. The platform (booking, CRM, dashboard) IS the product.
- **Not a Hallo Kira clone.** If we're just answering phones at €499/month, we're an overpriced version of Kira. The qualification depth, booking, and dashboard justify the premium.
- **Not a permanent state.** Phase 0 ends at 10 customers. After that, Phase 1 rules apply — higher prices, more templates, first hire, expansion preparation.
