## Phase 0 → Phase 1 Übergangskriterien

| Kriterium | Schwellenwert | Warum |
|-----------|-----------|-----|
| Zahlende Kunden | 10+ | Genug, um wiederholbare Verkäufe zu beweisen |
| MRR | €5K+ | Deckt grundlegende Gründerkosten |
| Rechtsgebiet-Templates | 3+ praxiserprobt | Onboarding ist effizient genug zum Skalieren |
| Anruf-Audit → Bezahlt-Konversionsrate | >40% | Vertriebsbewegung ist validiert |
| Kundenzufriedenheit | NPS >30 oder 3+ bereit als namentliche Referenzen | Produktqualität ist validiert |
| ElevenLabs CPA | Umgesetzt | Partnerschaftskanal ist aktiv |
| Fallstudien | 2+ mit namentlichen Kanzleien und echten Daten | Marketing-Assets für Phase 1 Outreach |

### Wenn Kriterien erfüllt sind:

1. Preise auf volle v4-Stufen anheben (Phase-0-Einrichtungsrabatte entfernen)
2. Erste Einstellung veröffentlichen (Customer Success)
3. Steuerberater-Outreach beginnen (Phase 2)
4. DATEV-Partnerschaftsantrag einreichen
5. RA-MICRO-Partnerschaftsgespräch starten
6. DAV Legal Tech Kooperationspartnerschaft beantragen

## Phase 0 → v4-Strategie-Ausrichtung

| v4-Strategieziel | Phase-0-Beitrag |
|-------------------|---------------------|
| Plattformunternehmen (eigenes IP, eigene Kunden) | Jede Installation läuft auf UNSERER Plattform, nicht nur ElevenLabs. Buchung, CRM, Dashboard = unser Burggraben. |
| Multi-Vertikal DACH-Expansion | Kanzlei-Templates übertragen sich zu 70-80% auf Steuerberater. Fallstudien übertragen sich als Social Proof. |
| ElevenLabs-Partnerschaft (Beschleuniger, keine Abhängigkeit) | Echter Nutzungsumsatz. Technische Referenz. Aber unsere Plattform hält die Kundenbeziehung. |
| RA-MICRO-Integrationsgraben | CSV-Export in Phase 0 → API-Integration in Phase 1 → tiefe Kooperationspartnerschaft in Phase 2. |
| €25K MRR in 12 Monaten | Phase 0 trägt €5-8K MRR bei. Phase-1-Ramp fügt €10-17K hinzu. Steuerberater-Expansion bringt den Rest. |

## Was Phase 0 NICHT werden darf

- **Kein Dienstleistungsunternehmen.** Wenn wir 20+ Stunden Custom-Arbeit pro Kanzlei leisten, ist das Modell kaputt. Templates müssen Onboarding reduzieren.
- **Kein ElevenLabs-Reseller.** Wenn unser einziger Wert „wir konfigurieren ElevenLabs für Sie" ist, sind wir eine Beratung. Die Plattform (Buchung, CRM, Dashboard) IST das Produkt.
- **Kein Hallo-Kira-Klon.** Wenn wir nur Telefone für €499/Monat annehmen, sind wir eine überteuerte Version von Kira. Die Qualifizierungstiefe, Buchung und das Dashboard rechtfertigen den Aufpreis.
- **Kein Dauerzustand.** Phase 0 endet bei 10 Kunden. Danach gelten Phase-1-Regeln — höhere Preise, mehr Templates, erste Einstellung, Expansionsvorbereitung.
