## Phase Gate Framework

### Phase 0 — Proof (now → 5 customers, ~90 days)

| Dimension | Detail |
|---|---|
| **Goal** | 5 paying customers, €10-15K MRR |
| **Pricing** | €0 setup, €2-3K/month, month-to-month |
| **Motion** | Warm network, niche-agnostic, founder does everything |
| **Product** | Full 7-agent platform demo. Voice is the wedge, but the pitch is agentic flows. |
| **Success** | 3+ customers who'd be upset if we disappeared + real before/after data |
| **Fundraise** | No. |

### Phase 1 — Repeatable (5 → 15 customers, ~6 months)

| Dimension | Detail |
|---|---|
| **Goal** | Repeatable sales, €30-50K MRR |
| **Pricing** | €5-12K setup, €2-5K/month |
| **Motion** | Case studies sell, inbound starts, patterns emerge |
| **Product** | Platform hardens through real usage |
| **Success** | New customers find us, not just us finding them |
| **Fundraise** | Only if capital accelerates something already working. |

### Phase 2 — Scale (15+ customers)

| Dimension | Detail |
|---|---|
| **Goal** | Own a category, expand verticals |
| **Pricing** | v3 tiers (€12K-€195K) become real |
| **Motion** | Add team, maybe raise from strength |
| **Success** | Revenue funds growth OR fundraise with leverage |

## What Phase 0 IS

- A bootstrapped proof-of-concept phase where every customer is a data point.
- SMB customers who close fast and deploy faster.
- Cheap, fast delivery — no procurement cycles, no legal review, no committee decisions.
- Founder does everything: sells, configures, supports, monitors, iterates.
- The minimum viable business that funds the maximum viable vision.

## What Phase 0 IS NOT

- The long-term strategy. This is a bridge, not a destination.
- Pricing we're proud of. €2-3K/month is intentionally below value to reduce friction.
- Scalable operations. Founder-led everything breaks at 10+ customers. That's fine — we only need 5.
- A pivot away from v3. The mid-market thesis stands. Phase 0 earns the right to execute it.

---

## How Phase 0 Feeds v3

Every SMB customer generates three things the mid-market strategy needs:

1. **Case study** — Real before/after data (missed calls, response times, appointments booked, revenue recovered).
2. **Battle-tested product** — Each deployment surfaces edge cases, tunes agent behavior, and hardens the platform.
3. **Testimonial** — A named business owner who says "this changed how we operate." Mid-market buyers need to see people like them already using it.

Phase 0 customers are the proof layer that makes v3 pricing credible. Without them, the €35K-€195K pitch is a slide deck. With them, it's a track record.

**Track A — Full Platform via Clara/Phone Wedge**
- Lead with Clara (Virtual Receptionist) as the entry point.
- Demo the full 7-agent platform. Voice answers the phone, but the pitch is the entire agentic operations layer.

**Track B — Platform via Kai/Vacation Ops Wedge**
- Lead with Kai (Team Operations) as the entry point.

- ICP: businesses with 10+ employees and coordination overhead.
- Documented separately in `/docs/strategy/one-of-one-v3/phase-0-kai/`.

Both tracks share phase gates. Combined customer count and MRR determine phase transitions.

## Honest Constraints

- **Solo founder.** Every hour is either building or selling. There is no third category.
- **Zero budget.** No paid ads, no SDRs, no conference booths. Network and hustle only.
- **Zero customers.** No social proof, no case studies, no references. The demo has to carry the weight.
- **Zero revenue.** Personal runway is the only runway. Phase 0 pricing must generate cash fast.
- **Time is the bottleneck.** Building and selling compete for the same hours. The weekly rhythm (see `02_PROOF_PHASE_SALES_MOTION.md`) is the constraint that makes this work.
