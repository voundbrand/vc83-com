## Sales Process

### **Self-Service Funnel (80% of customers)**

```
Traffic ‚Üí Landing Page ‚Üí Sign Up ‚Üí Onboarding ‚Üí Activation ‚Üí Paid
```

**1. Landing Page**
- Hero: Value prop + visual demo
- Social proof: Testimonials, logos, metrics
- Comparison: vs. GHL native
- Pricing: Transparent, clear tiers
- CTA: "Start Free Trial" (14 days, no credit card)

**2. Sign Up**
- Email + password (no friction)
- OAuth optional (for easier GHL connection)
- Skip payment info (lower barrier)

**3. Onboarding**
- Guided wizard (Step 1-6 from implementation doc)
- Video tutorial embedded
- "Send test message" within first 5 minutes
- Success: Confetti + "You're all set!"

**4. Activation**
- Goal: 10+ conversations in first 7 days
- Trigger: Email if < 5 conversations by Day 3
- Incentive: "Set up your first agent and get a free month"

**5. Conversion to Paid**
- Day 10 email: "Your trial ends in 4 days"
- Day 13 email: "Last day of trial! Here's what you've achieved..."
- Day 14: Prompt to add payment info
- Offer: "Start annual plan and save 20%"

### **High-Touch Sales (20% of customers)**

For $500+/mo deals (agency tier, white-label, enterprise):

```
Inbound Lead ‚Üí Discovery Call ‚Üí Demo ‚Üí Proposal ‚Üí Close
```

**1. Lead Qualification**
```
Questions to ask:
- How many sub-accounts do you manage?
- What's your current DBR process?
- What tools are you using today?
- What's your budget for AI/automation tools?
- When are you looking to implement?
```

**2. Discovery Call (30 minutes)**
```
Agenda:
- Understand their DBR challenges (10 min)
- Show memory engine in action (10 min)
- Discuss pricing and ROI (5 min)
- Next steps and timeline (5 min)
```

**3. Custom Demo (if needed)**
```
For larger deals:
- Set up with their actual GHL data
- Run real reactivation scenarios
- Show memory layers with their conversations
- Calculate ROI based on their numbers
```

**4. Proposal**
```
Include:
- Custom pricing (volume discount)
- Implementation timeline
- Success metrics
- Support SLA
- Contract terms (monthly or annual)
```

**5. Onboarding**
```
White-glove for high-value customers:
- Dedicated onboarding call
- Help configure first 5 agents
- Weekly check-ins for first month
- Direct Slack/email support
```
