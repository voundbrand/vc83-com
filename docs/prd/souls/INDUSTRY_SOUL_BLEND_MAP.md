# Industry Soul Blend Map — The Book of 100 AI Agents

## How This Document Works

Each of the 98 industry agents is mapped to 2-4 real people whose frameworks, philosophies, and methodologies create a **hybrid soul**. The blend percentages indicate how much of each person's thinking dominates the agent's behavior.

**Selection criteria for soul candidates:**
- Published frameworks with documented methodology
- Distinctive voice and communication style
- Domain authority with real-world proof
- Complementary blendability (not redundant)
