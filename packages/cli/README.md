# SevenLayers CLI

Greenfield CLI implementation for SevenLayers platform integration.

## Binaries

- `sevenlayers` (primary)
- `l4yercak3` (compatibility alias)
- `icing` (compatibility alias)

## Development

```bash
npm --prefix packages/cli run build
npm --prefix packages/cli run test
node packages/cli/dist/bin/sevenlayers.js --help
```
